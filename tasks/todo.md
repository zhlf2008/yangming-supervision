# 任务：重新设计学委课程合集卡片

## 设计目标

在保持系统整体风格（暖色调、宋体/衬线字体）的前提下，提升课程卡的视觉品质和信息层级，使其更精致、更专业。

## 当前问题

- 封面区域简单，占位符只有 emoji，缺乏视觉吸引力
- 卡片 body 信息层级平淡（标题→描述→章节数→操作按钮一字排开）
- 操作按钮占用空间大，文字按钮视觉噪音多
- 整体缺乏层次感和细节打磨

## 设计方案

### 1. 卡片容器
- 微调圆角和阴影，增加 `backdrop-blur` 质感
- hover 时 shadow 变化更细腻，去掉 `translateY` 改用 `box-shadow` 深度变化

### 2. 封面区域（核心改动）
- 封面图下方增加半透明渐变遮罩（从底部向上渐变）
- 封面图上叠加显示课程章节数（白色半透明标签）
- 无封面时，占位区域使用更丰富的渐变色 + 书籍图标
- 封面图片 hover 时 `scale(1.05)` 平滑放大（`overflow: hidden` 裁剪）

### 3. 卡片 Body
- 标题加大加粗（16px → 17px），行距优化
- 描述改为用 `color: var(--ink-light)` 而非 muted，提高可读性
- 章节数用圆角 pill 标签形式展示，放在标题右侧（flex 布局）
- 卡片底部区域紧凑排版操作按钮（图标 + 文字，更小巧）

### 4. 操作按钮
- 编辑/停用改为小型按钮，停用按钮用图标替代（非激活时绿色启用）
- 使用更精致的按钮样式（小圆角、小号 padding）

### 5. 动画细节
- 卡片入场添加淡入动画（CSS animation）
- hover 封面图片缩放平滑过渡

## 改动范围

仅修改 `study-course-library.html` 文件：
- `<style>` 块中的 `.course-card*` 相关 CSS（约 10 条规则重写）
- `renderCourses()` 函数中的卡片 HTML 模板（重写模板字符串）

## 影响评估

- ✅ 仅改动卡片渲染部分，不影响数据逻辑
- ✅ 不修改 JS 业务函数（loadCourses/saveCourse/editCourse/toggleCourse）
- ✅ 不修改对话框/表单区域
- ✅ 兼容已有封面图 URL 数据
